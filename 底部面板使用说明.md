# 底部面板使用说明

## 功能概述

底部面板已成功实现并集成到Outlook插件中。由于Microsoft Outlook VSTO的技术限制，我们采用了浮动面板的解决方案。

## 使用方法

### 1. 显示/隐藏面板
- 在Outlook功能区的"MyList"选项卡中，点击"底部面板"按钮可以显示或隐藏面板
- 面板默认为隐藏状态

### 2. 最小化/还原面板
- 点击"最小化"按钮可以最小化面板内容
- 再次点击可以还原面板内容

### 3. 调整面板位置
- 面板采用浮动模式，您可以通过拖拽标题栏将面板移动到任意位置
- **推荐位置**：将面板拖拽到主邮件阅读区域的下方，这样不会遮挡左侧导航栏和右侧日历/待办事项区域
- 面板会记住您设置的位置

### 4. 调整面板大小
- 您可以通过拖拽面板边缘来调整面板的大小
- 默认尺寸：宽度600像素，高度200像素

## 技术说明

### 为什么使用浮动面板？

Microsoft Outlook VSTO的CustomTaskPane只提供以下5种固定停靠位置：
- `msoCTPDockPositionLeft` - 停靠到左侧
- `msoCTPDockPositionRight` - 停靠到右侧  
- `msoCTPDockPositionTop` - 停靠到顶部
- `msoCTPDockPositionBottom` - 停靠到底部（占用整个底部区域）
- `msoCTPDockPositionFloating` - 浮动模式

由于`msoCTPDockPositionBottom`会占用整个Outlook窗口的底部区域（包括左侧导航栏和右侧日历区域下方），无法实现只在主邮件区域下方显示的需求。

浮动面板是最佳的替代方案，它提供了最大的灵活性，用户可以根据自己的使用习惯将面板放置在最合适的位置。

### 面板功能

- **主题适配**：面板会自动适配Outlook的明暗主题
- **状态记忆**：面板的显示/隐藏状态和最小化状态会被记住
- **响应式设计**：面板内容会根据大小自动调整布局

## 建议的使用方式

1. 首次使用时，显示面板后将其拖拽到主邮件阅读区域的下方
2. 调整面板大小以适合您的屏幕和使用习惯
3. 根据需要使用最小化功能来节省屏幕空间
4. 面板位置和大小设置会在Outlook重启后保持

## 注意事项

- 浮动面板可能会遮挡其他内容，请根据需要调整位置
- 如果面板消失或位置不当，可以通过"底部面板"按钮重新显示并调整
- 面板的具体功能和内容可以根据后续需求进行扩展
